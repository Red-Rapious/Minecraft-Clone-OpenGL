# Minecraft-Clone-OpenGL
A clone of the famous java game, coded in C++ with OpenGL. Probably my biggest, most personal project yet

## Libraries used :
- `GLFW`
- `GLAD` (new versions)
- `GLEW` (for older versions of the repo)
- `glm` for maths
- `stb_image` for png loading
